# Frontend (React + Vite)

Install dependencies and run the dev server (PowerShell):

```powershell
cd "c:\Users\Kartikeyan Dubey\Desktop\ecomm part 2\frontend"
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` by default. The frontend expects the backend API to be available at `/api/...` (use a proxy or run backend on another port and configure CORS).
